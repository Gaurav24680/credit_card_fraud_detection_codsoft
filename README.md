Credit Card Fraud Detection is a machine learning project developed to identify fraudulent credit card transactions. With the rapid growth of online transactions, fraud detection has become an important challenge for financial institutions. This project uses machine learning algorithms to classify transactions as either fraudulent or legitimate based on transaction-related features.

The main objective of this project is to build an accurate and reliable fraud detection model using historical transaction data.
🚀 Features
Data preprocessing and cleaning
Handling missing and categorical values
Exploratory Data Analysis (EDA)
Fraud vs Legitimate transaction analysis
Machine Learning model training
Performance evaluation using classification metrics
Fraud prediction for new transactions
Visualization of results

####Technologies Used
Python
Google Colab / Jupyter Notebook
Pandas – Data manipulation
NumPy – Numerical operations
Matplotlib & Seaborn – Data visualization
Scikit-learn – Machine learning models
